# Frontend

Static frontend for DigiShop.

## Run Locally

Open `index.html` directly in a browser, or serve the folder:

```bash
python3 -m http.server 5500
```

Then visit `http://localhost:5500`.

## Pages

- `index.html` - role selection gateway
- `login.html` and `signup.html` - authentication UI
- `customer.html`, `cart.html`, `payment.html` - customer flow
- `shopkeeper.html`, `shop_register.html` - shopkeeper flow
- `delivery_login.html`, `delivery_dashboard.html` - delivery partner flow
- `admin.html` - admin dashboard mockup
